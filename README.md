## Install

You can download prebuilt binaries from the GitHub releases or from our [download mirrors](http://download.owncloud.com/ocis/MARKDOWN-EDITOR/). For instructions how to install this on your platform you should take a look at our [documentation](https://owncloud.github.io/ocis-markdown-editor/)

## Development

Make sure you have a working Go environment, for further reference or a guide take a look at the [install instructions](http://golang.org/doc/install.html). This project requires Go >= v1.13.

```console
git clone https://github.com/owncloud/ocis-markdown-editor.git
cd ocis-markdown-editor

make generate build

./bin/ocis-markdown-editor -h
```
